## Les grandes nouveautés par domaine

### 🎫 Tickets — un module ITSM complet
Le cœur de la plateforme s'est étoffé en un véritable outil de gestion de services.
- **Module ITSM complet** : KPI historisés, tri des colonnes, vues Table / Kanban / Inbox (scroll infini).
- **SLA** : définitions, calendriers ouvrés, suivi des dépassements, bouton de réinitialisation (purge + recalcul) et table de conversion jours ouvrés → minutes.
- **Règles d'affectation** : actions automatiques (`set_vip`, `boost_priority`, `set_type`, `set_category`, `add_tag`), onglet VIP, icônes de catégorie.
- **Workflow** : bouton « Solutionner » (commentaire → solution + Résolu), filtre Type (Incidents / Demandes), classification des tickets live avant clôture.
- **Réponses automatiques**, **base documentaire (KB)** et parcours d'aide complet, docs MagApp rattachés au logiciel.
- **Journal** séquentiel de l'historique d'un ticket, page de **paramètres configurables** (chat, couleurs, AD, clôture auto, dictée vocale).
- **Clôture automatique** des tickets résolus (délai paramétrable, cron de minuit, log des clôtures auto + demandeurs) et affichage des résolus par défaut.

### ✅ Mes Tâches — agrégation multi-sources
- **Module unifié** agrégeant les tâches de tous les modules (tickets, projets, réunions, rencontres, transcripts, MS To-Do).
- **API de tâches d'équipe unifiée** + `AddTaskModal` réutilisé partout (projets, réunions…).
- Présentation en tableau, **notes & pièces jointes**, synchronisation **MS To-Do**, tuile à 3 badges.
- **Tâches d'équipe** : tâche unique partagée, statut commun (le premier qui la termine la termine pour tous), liste des destinataires, notification du créateur en cas de refus, **mail à l'affectation** et édition propagée au groupe.
- Alertes email quotidiennes (opt-in) + bascule de préférence.

### 📁 Projets & Gouvernance — portefeuille PMO
- **Module Portefeuille Projets** : schéma PostgreSQL, API REST, interface React.
- **Sous-projets** indentés sous le parent, colonnes triables, rupture visuelle par niveau d'implication.
- **Gouvernance** : comités/comitologie, chef de projet métier (AD), membres de comité, cases DPD/RSSI, recherche AD.
- **PMO** : visibilité sur tous les projets, filtrage par périmètre, groupement par chef de projet, tri par implication.
- **Étapes personnalisables** par projet avec fil d'Ariane et filtre workflow.
- Applications liées, projet parent, types documentaires, upload de fichiers, gestion des transcripts (MeetingDetail).
- **Réunions** : page « Mes réunions », modale de détail directe, comités, heure de création (datetime-local), commentaires participants, PJ dans les CR, ajout auto du créateur.

### 🖨️ Copieurs — gestion de flotte SNMP
- **Module complet** : CRUD, ping, déplacement, géolocalisation/carte avec clustering, import PaperCut.
- **Collecte SNMP automatique** (toners, erreurs, compteurs) avec **cron quotidien à 11h**, test SNMP direct, SNMP Walk de diagnostic, barre de progression.
- **Relevés détaillés Canon** (A4/A3, N&B/couleur), mapping SNMP Koesio, gestion compteurs/tarifs/relevés.
- **Imports Excel** (Canon trimestriel, annuel reconstruit, refonte par mainteneur/année).
- **Tableau de bord KPI** : évolution des copies, coûts, alertes dégressives, projection annuelle au 31/12 avec graphiques (barres hachurées, lignes pointillées).

### 💻 Parc informatique, Déploiements & Mobilité
- **Module de parc live/hub** synchronisé depuis **GLPI 10** (schéma `hub_parc`, 4 tables).
- **Tableau de bord** remanié : pyramide des âges, widgets DSI, tri des colonnes, onglet Géo, infobulles de détail, filtres par statut.
- **Déploiements** : import `deploy.xlsx`, rapprochement avec le parc/AD/GLPI, détection d'incohérences, colonne Équipement à type inféré (PC, écran, bundle…), édition de fiche, visionneuse PDF/image/DOCX, **graphiques de cadence** (par année / mois).
- **Mobilité** : onglet Rebut, listes stock/rebut, édition d'appareils.
- Recherche AD, inversion usager/n° usager, email dans la liste.

### 🔄 GLPI — synchronisation
- **Onglet GLPI 10** unifié (token d'API unique) dans les Synchronisations.
- Sync « récents » enrichie (50 derniers tickets + descriptions + commentaires) et **synchro complète en une passe** (tickets + descriptions + suivis en parallèle).
- Synchro des **noms/prénoms** des demandeurs et observateurs via API GLPI + AD.

### 💶 Budget, Finance & Oracle
- **Module d'automatisation Oracle** : scheduler, logs, stockage des synchros, tables de paramètres (RH & Finances).
- **Migration commandes/factures vers PostgreSQL**, colonnes dynamiques, onglet Tiers intégré.
- **Field mapping** + `MappedDataTable`, expressions arithmétiques, filtre factures.
- **Opérations** : badge du nombre de commandes, déroulé des commandes liées, KPI réalisé fonctionnement/investissement, total prévu et % de réalisation.

### ✉️ Mail, Notifications & Collecteur
- **Mail Collector** : création automatique de tickets depuis les e-mails O365 (logs, routage, fréquences).
- **Module d'automatisation des e-mails** + onglet alertes tâches + journal global des envois.
- Consolidation des réglages **O365 / Azure AD**, file de notifications, pagination des logs, **kill-switch** pour stopper tous les envois.
- Éditeur HTML de templates, recherche de l'e-mail via AD en priorité pour les notifications de changement de statut.

### 📝 Backlog & « What's New »
- **Système de backlog** : formulaire de soumission, interface admin, pièces jointes, sélection module/tuile, commentaires admin, archivage.
- Édition titre/demandeur avec recherche AD, regroupement par statut, mise en avant des demandes de l'utilisateur.
- **Version release** depuis l'admin backlog + page **« Nouveautés »** publique.

### 💬 Live & Chat
- **Sessions live** : pré-clôture (demandeur/timeout) puis finalisation technicien, classification Incident/Demande, template `live_summary`.
- Chat-frontend conteneurisé (Docker DMZ) lisant le branding depuis la config admin, EmojiPicker, panneau des sessions.
- **Alertes d'urgence** : contacts mobile/urgence d'équipe, configuration WhatsApp.

### 🌐 Réseau / Infra, Stocks & autres modules
- **Module Réseau/Infra** : carte des switchs et liens, dashboard, API, schéma, sites filtrés par équipement, couche cœur.
- **Module Stocks** : workflow livraison/BL complet, modèles de BL, stats/KPI, transposition.
- **Inventaire** : dashboard hôtes/alertes/incidents/vulnérabilités/SIEM.
- **Calendrier DSI** : refonte UI, intégration O365 (auto-sync, mapping catégories), vue jour multi-colonnes, couleur DSI.
- **Dashboard DSI** : tuiles déplaçables avec ordre persistant, badges contrats expirés/bientôt expirés.
- **Consommables** (organigramme, écoles, récap annuel), **Contrats**, **Certificats** (visionneuse PDF + accès SMB en conteneur), modules **Ville** et **Transcripts** (partage par direction/service).
